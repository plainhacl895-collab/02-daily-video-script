# 视频 CTA 策略

视频里绝不流露任何"找我买房"的暗示——观众对中介高度敏感，一闻到推销味就划走。

**短期目标**：点赞 + 关注 + 收藏 → 把账号和人设做起来、粉丝量做起来
**长期目标**：粉丝 → 信任 → 自然转化客户

**Why:** 用户指出这类账号太多，观众敏感度极高。即使说"买不买无所谓"，观众也会解读为中介在遮掩推销意图。必须先让账号有价值、有人设、有粉丝基础，转化在内容之外自然发生。

**How to apply:** 每条脚本的结尾 CTA，只引导点赞/关注/评论互动，不问"有需求吗？""我帮你分析""要不要看看"。内容提供价值，人设自然吸引客户，不在视频里做成交动作。
