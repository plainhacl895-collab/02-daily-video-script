# 阶段性复盘记录

## 复盘节奏

- **验证期（第1-8周）**：每2周一次
- **深耕期（第9-24周）**：每4周一次
- **破圈期（第25周+）**：每4周一次 + 每季度一次深度复盘

---

## 复盘模板

```markdown
### 第X周复盘 — YYYY-MM-DD

**当前阶段**：[验证期/深耕期/破圈期]

**数据总结**：
- 本周期发布X条，完播率均值X%，互动率均值X%
- 最佳单条：[标题] 完播率X% 互动率X%
- 来自视频的有效私信：X条
- 新增客户咨询：X个

**内容发现**：
- [哪种类型/话题/风格表现最好？有数据支撑吗？]
- [哪种表现明显不好？]

**竞品动态**：
- [赛道里有什么值得关注的变化？]

**策略调整**：
- 做了什么调整？
- 依据是什么？
- 下次复盘需要验证什么？

**定位修正**（如果有）：
- 是否需要更新 [[positioning]]？
```
