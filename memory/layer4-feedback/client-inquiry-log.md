# 客户咨询来源记录

> 这是整个记忆系统里最重要的一份数据。它直接回答：**"我做视频到底有没有用？"**

## 记录格式

| 日期 | 来源视频 | 客户简称 | 预算 | 目标板块 | 需求类型 | 状态 | 备注 |
|------|---------|----------|------|---------|----------|------|------|
| | | | | | | | |

### 字段说明
- **来源视频**：客户说是从哪条视频来的（或你判断的）
- **预算**：客户报的预算范围
- **需求类型**：改善置换 / 首次入市 / 资产配置 / 学区 / 其他
- **状态**：跟单中 / 已带看 / 已成交 / 已流失
