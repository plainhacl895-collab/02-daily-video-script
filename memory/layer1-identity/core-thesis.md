# 核心观点库

## 什么是"好观点"

- 是可争论的判断（有人说"对"也有人说"不对"），不是正确的废话
- 来自真实交易经验，不是网上搬运
- 每条能展开成至少3条完整视频选题

## 待验证观点（想做但还没做的）

- [ ] 1000-1500万的改善客户，最终决策往往不是比价格，是比"哪个中介让我更放心"
- [ ] 视频号和小红书的房产内容不缺分析型，缺陪伴型——这个缺口就是差异化空间
- [ ] 直播看房不如短视频建立信任有效，因为高净值客户不需要"看房"，需要"找到对的人陪他看房"
- [ ] 中介同行做视频最大的通病是"急于证明自己专业"，而客户真正要的是"让我感觉被理解"
- [ ] 每套房子都有缺点，敢于说出缺点的中介反而成交率更高——因为你是第一个不骗他的人
