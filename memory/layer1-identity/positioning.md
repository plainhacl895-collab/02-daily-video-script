# 定位决策记录

## V1 — 初始定位（2026-05-15）

### 赛道
- 领域：上海改善置换经纪人
- **核心板块**：长宁 · 天山 / 古北 / 中山公园（距离最近、最深耕的根据地）
- **拓展板块**：普陀 / 闵行 / 静安（周边外延，保证客户池够大）
- 板块策略：以长宁三个板块为专业深度，周边区为广度覆盖。不装全上海都熟，但也不把自己困死
- 预算锚点：**1000万-1500万**（核心区间），上限约2000万，800万以下不碰
- 客群画像：**待填**
- 交易类型：**待填**

### IP方向权重（1-10分）
- 行家型（数据/板块/逻辑）：4
- 避坑型（揭露问题/说真话）：4
- 陪伴型（客户故事/情感共鸣）：9
- 资源型（独家渠道/搞定能力）：4

### 一句话定位
> 我是帮上海改善家庭，把复杂的买房决策拆成一步一步看清楚，不催单、不推销，但每一句都有依据。

### 差异化来源（2026-05-15 挖掘对话记录）
- **核心特质**：专注、说实话、不夸大
- **核心矛盾**：说实话可能吓跑客户 vs 不说违背自己 → 解法：把真相翻译成决策依据，帮客户在真相里找到最优解
- **同行反感**：吹牛、夸大其词、只为成交骗客户
- **内容载体**：陪伴型（9分），不是推销员是那个跟你说实话、陪你看清选项的人
- **专业表达**：不是"数据碾压"的行家型，而是"把复杂决策拆成一步一步看清楚"的方法论型专业

### 核心价值观（已从 persona.json 继承）
- 不催单，帮客户筛选而非推销
- 用真实数据说话，不编造
- 承认不完美，没有十全十美的房子
- 站在客户立场想问题

### 金句（已继承）
- 我不帮你做决定，我帮你看清选项。
- 好房子不是推销出来的，是筛选出来的。
- 买不买无所谓，先让你知道真实情况。
- 说到底，房子是给人住的，合适比便宜重要。
