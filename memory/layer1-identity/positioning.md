# 定位决策记录

## V2 — 知识型转型（2026-05-23）

### 转型原因

陪伴型博主门槛太低，市场饱和。观众对中介高度敏感，任何推荐都被解读为带货。只有用**信息量和判断力**建立话语权，才能让观众从"你在推销"变成"你在教我"。

核心逻辑：知识型内容 = 筛选高素质受众 + 建立不可替代的专业壁垒 + 长尾搜索价值。

### 赛道
- 领域：上海板块/小区深度分析
- **核心板块**：长宁 · 天山 / 古北 / 中山公园（最深耕的根据地，以此为起点建立"板块专家"认知）
- **拓展板块**：普陀 / 闵行 / 静安（逐步覆盖，每新增一个板块至少做5条深度内容）
- **内容系列化**：一个板块一个系列，每个系列覆盖板块内核心小区。观众看完一个系列就能独立判断。
- 预算锚点：**1000万-1500万**（核心区间），上限约2000万，800万以下不碰
- 客群画像：外地父母给上海的孩子买房为主
- 交易类型：改善置换

### IP方向权重（1-10分）
- 行家型（数据/板块/逻辑）：**9** ← 核心卖点
- 避坑型（揭露问题/说真话）：4
- 陪伴型（客户故事/情感共鸣）：3
- 资源型（独家渠道/搞定能力）：3

### 一句话定位
> 用数据和实地调研，把上海每个板块的真实情况讲透。不推荐楼盘，只提供判断依据。

### 差异化来源
- **核心壁垒**：系统性的一手小区数据 + 实地探盘经验。不是"我看过这个板块"，而是"这个板块11个小区我每个都走过、每个都有数据"。
- **不可复制**：板块级的知识积累需要时间和腿力，同行即使想抄，没有实地踩过盘也讲不出细节。
- **表达风格**：保持朋友聊天式的语气（不生硬上课），但内容骨架是数据和实地调研。用"我最近踩盘发现"而不是"我来教你"。
- **信任建立**：不推荐楼盘，不引导成交。视频里只做一件事——提供判断依据。观众自己得出结论。

### 核心价值观（继承 V1）
- 不催单，帮客户筛选而非推销
- 用真实数据说话，不编造
- 承认不完美，每个小区都有缺点
- 站在客户立场想问题
- 把复杂的决策拆成一步一步看清楚

### 金句（继承 + 新增）
- 我不帮你做决定，我帮你看清选项。
- 好房子不是推销出来的，是筛选出来的。
- 买不买无所谓，先让你知道真实情况。
- 说到底，房子是给人住的，合适比便宜重要。
- **新增**：不讲哪个小区好，讲它适合谁、不适合谁。

### 知识型内容的四条铁律
1. **每期必须有可验证的数据**——均价、成交量、房龄分布、户型占比，至少给出3个具体数字
2. **每期必须给出判断**——不只是罗列数据，而是"这个数据意味着什么"
3. **每个小区必须讲缺点**——没有十全十美的小区，回避缺点就是推销
4. **不推荐，只分析**——"适合XX类型的人"代替"推荐买这里"
