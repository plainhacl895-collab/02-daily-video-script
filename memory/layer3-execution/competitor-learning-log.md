# 竞品视频拆解学习日志

> 每次发来视频拆解后，把可复用的技法记在这里。不改动现有技能文件。
> 积攒够 10+ 条后，统一分析：哪些要纳入系统？哪些要改编？哪些不适用？

## 技法评估框架

每条学习记录标注三个维度：

| 维度 | 说明 |
|------|------|
| **可直接拿** | 写作/制作技法，不涉及人设风格，直接用 |
| **需改编** | 框架思路可用，但表达方式要换成佳佳风格 |
| **不适合** | 跟佳佳定位/受众冲突，放弃了但记录原因 |

## 学习记录

### 2025-05-19 — 上海小光头《小院生活》

**视频数据**：点赞 3081 | 收藏 882 | 分享 1154 | 时长 3分16秒
**账号**：上海小光头丨造家计划（748条作品）
**视频链接**：https://v.douyin.com/0aRhyixpSbA/

**评论区关键洞察**（10条中4条夸文案）：
- 58赞："文案解说真的一级棒，更多可能还是听听文案"
- 8赞："文案真的棒，我决定抄一下发朋友圈"
- 0赞："房子一般，解说非常好"
- 结论：观众来评论区夸文案，说明文案本身成了内容

**提取技法**：

| # | 技法 | 评估 | 说明 |
|---|------|------|------|
| 1 | **材质名词替代形容词** — 不说"高级"，说"黑胡桃木地板+高光水性漆+釉面质感+极窄边框玻璃" | 🟢 可直接拿 | 纯写作技法，不涉及人设。形容词是下结论，名词是让观众自己下结论 |
| 2 | **一个空间一个记忆点** — 探盘时每个空间只让观众记住一件事，不罗列参数 | 🟢 可直接拿 | 信息组织方式，跟人设无关。适合实拍/探盘类视频 |
| 3 | **"认怂钩子"**（先抑后扬）— 先替观众说出"嫌弃"，再反转制造稀缺。原句："2000年的房子，既没有老房子的韵味，又没有新房的配套，优势在哪？" | 🟡 需改编 | 框架可以拿，但表达要改。佳佳版应为"说实话这套房不完美，但有一个点很值得聊"——保持朋友聊天式，不要朗诵感 |
| 4 | **生活方式宣言收尾** — "最爱的电影，最爱的whiskey，最爱的家" | 🟡 需改编 | 价值观收尾的思路已经在系统里（技巧5），但表达方式要换成佳佳版："买不买无所谓，先让你知道真实情况" |
| 5 | **品牌奢侈品类比** — 用爱马仕铆钉工艺类比橱柜 | 🔴 不适合 | 佳佳受众是务实改善家庭（1000-1500万），不是奢侈品消费者。过度品牌类比会产生距离感 |
| 6 | **诗意的文学腔调** — 整条视频是朗诵式演绎，强调画面美学 | 🔴 不适合 | 跟佳佳"朋友聊天式"语言风格冲突。佳佳的内容价值在专业判断，不在文学感染力 |
