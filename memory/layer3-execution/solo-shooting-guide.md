# 单人拍摄实操指南

> 你是单人操作，没有拍摄团队。这份指南帮你用最低成本高效产出三种形式的视频。

## 设备方案（三级）

### Tier 1 — 零成本起步（第1-2周）

- **拍摄**：近3年内的任意智能手机，相机设为 4K 30fps
- **固定**：任意桌面三脚架（<50元），或用书堆垫到眼睛高度
- **收音**：手机自带麦克风，距离30-50cm，在安静房间录制（地毯/窗帘吸回声）
- **灯光**：纯自然窗光，面朝窗户（绝不背光），拍摄时间 9am-11am 或 3pm-5pm
- **剪辑**：手机版剪映免费版，前5条可以先用"一键成片"模板上手

### Tier 2 — 收到第一条有效私信后投资（第3周+）

- **麦克风**：Boya BY-M1 有线领夹麦（~80元），或一步到位 Rode Wireless GO II（~1500元）
- **灯光**：Godox SL-60W LED面板灯（~400元），阴天也能保持稳定光线
- **三脚架**：高点手机三脚架 + 遥控快门（~100元）
- **稳定器**：DJI OM 6（~800元），实拍探盘必备，消除走动抖动

### Tier 3 — 稳定出单后升级

- **相机**：Sony ZV-E10 或 iPhone 15/16 Pro + 稳定器
- **麦克风**：DJI Mic 2 无线系统
- **监看**：Feelworld 外接监视器（探盘时确认构图）

## 三种形式的单人拍摄步骤

### 口播

**拍摄前（10分钟）**：
1. 脚本放手机分屏或iPad上，位置紧贴摄像头下方（不要放侧面，视线偏移会被看出来）
2. 换一件纯色上衣（避开细条纹/密集格子，镜头前会产生摩尔纹）
3. 坐下或站立均可，但一旦选了就不要中途换姿势

**拍摄中（20分钟，3-4条）**：
1. 第一条当热身，第三条通常是最好的。NG了不用整条重拍——暂停、呼吸、从那段开头重来，后期剪掉卡顿
2. 说话能量 = 跟一个朋友打电话，不是给一个教室上课。音量 = 正常对话。语速 = 比日常稍慢半拍
3. 一次录3-4条脚本，中间换件外套/换个背景角度，制造"不同天"的错觉

**拍摄后（15分钟剪辑）**：
- 剪映操作：导入 → 剪掉卡顿和长停顿 → 自动字幕 → 关键数字/金句处加大字幕叠层 → 加BGM（音量调到30%）→ 导出

---

### 实拍/探盘

**拍摄前（20分钟到场准备）**：
1. 打开所有窗帘，打开所有灯，听一下外面有没有施工噪音
2. 把房东私人物品移出镜头（桌上杂物、晾衣架、个人照片）
3. 规划拍摄路线：门口开场 → 客厅 → 厨房 → 卧室 → 阳台 → 最佳位置收尾

**拍摄中（40分钟，两遍法）**：
- **第一遍（自拍模式）**：你对着前置镜头讲，边走边说，这是主音轨
- **第二遍（B-roll）**：每个房间稳定平移镜头5-10秒，拍地面/柜门把手/窗景/层高感受的细节镜头
- 不要试图一遍同时做好——单人操作会手忙脚乱
- 室内有人时告知在拍摄，有人时用紧角度避开人脸

**安全红线**：
- 绝不公开精确地址或能识别具体门牌号的建筑外立面
- 房东未同意不出镜任何家庭成员

**拍摄后（30分钟剪辑）**：
- 对齐VO音轨到B-roll画面 → 房间标注文字叠层（"客厅 朝南 采光"）→ 结尾加联系引导 → 导出

---

### 混剪

**拍摄前（5分钟）**：
- 准备素材：3-5张链家/贝壳截图、1张高德地图截图、1-2张数据表、2-3张房源照片
- 脚本打印或放屏幕上

**录制VO（15分钟）**：
- 在安静房间一次录完整个脚本
- 边录边在脚本上标记时间戳，标注哪里需要插入什么画面

**剪辑（25分钟）**：
- 剪映操作：放入VO音轨 → 自动字幕 → 根据标记在时间点插入素材图片 → 每张素材停留3-5秒 → 加转场（淡入淡出即可，不花哨）→ 加BGM（音量25%）→ 导出
- 第一次做好模板后存为"房产分析模板"，以后复用只替换素材和VO

## 每周拍摄节奏建议

| 日期 | 形式 | 耗时 | 说明 |
|------|------|------|------|
| 周一 | 口播 ×2 | 1.5h | 一次录两条，换装 |
| 周二 | 混剪 ×1 | 1h | 用周末整理的数据 |
| 周三 | 实拍 ×1 | 2h | 尽量结合真实带看，一举两得 |
| 周四 | 口播 ×1 | 45min | 故事/陪伴型 |
| 周五 | 混剪 ×1 | 1h | 本周总结/清单式 |
| 周六 | 实拍或休息 | 2h | 周末看房顺便拍 |
| 周日 | 休息 | - | - |

**周产量**：6-7条视频，约8小时总耗时。全职经纪人可承受。

## 快速自查（每次拍摄前）

- [ ] 手机存储空间够吗？（4K视频约400MB/分钟）
- [ ] 镜头擦干净了吗？（手指油渍是最常见的画质杀手）
- [ ] 麦克风插好了吗？（录了3分钟发现没插麦=废片）
- [ ] 光线是面朝窗户而不是背对窗户吗？
- [ ] 背景里有没有不该出现的东西？（个人物品/反光面/乱堆的杂物）
- [ ] 脚本提前读过3遍了吗？（不要读稿感，熟悉到能自然说出来）
