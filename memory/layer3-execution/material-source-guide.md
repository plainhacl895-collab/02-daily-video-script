# 素材来源与捕捉指南

> 你不是缺素材，你是缺一个"把日常工作翻译成视频选题"的习惯。
> 房产中介每天接触的信息量，足够支撑每天一条视频。

---

## 核心认知

拍视频最耗脑子的不是写脚本，是**选题材**。但对你来说，素材不是"想"出来的，是**从日常工作中捡出来的**。

你每天已经在做的事：
- 跟客户聊天 → 客户的疑问 = 选题
- 带看房源 → 房子本身 = 画面素材 + 分析素材
- 刷贝壳/链家 → 数据变化 = 选题
- 看同行发的内容 → 启发（不是抄）= 角度素材
- 客户成交/不成交 → 故事 = 选题
- 路过一个小区 → 观察 = 话题素材

**一句话原则**：你今天工作中遇到的任何一件小事，都可以变成一条视频。

---

## 七大日常素材来源

### 1. 客户一句话 → 一条视频

客户问过的每一个问题，都有至少 100 个潜在客户也在问。

**捕捉方式**：
- 每次跟客户聊完天，回想一句他说的原话
- 记在微信文件传输助手里，就一句话，不用整理
- 格式：`日期 + 客户原话 + 你怎么回的`

**例子**：
| 客户原话 | → 视频选题 |
|----------|-----------|
| "天山和古北挨着，为什么单价差3万？" | 《天山vs古北：隔一条街，差价3万到底差在哪》 |
| "1000万在上海能买到什么样的房子？" | 《1000万预算，长宁三个板块真实能买到什么》 |
| "这个户型看起来挺好，你为什么说不建议？" | 《有一种户型，看着没毛病，住进去就想换》 |
| "现在买会不会站岗？" | 《现在入手改善房，会不会买在最高点？》 |
| "房东为什么卖这个价？能砍多少？" | 《房东报价的逻辑：哪些能砍、哪些砍不动》 |

> 客户问过你的问题，就是最好的选题。因为这个问题真实存在，而且有明确受众。

### 2. 今天带看的房子 → 实拍素材

只要进了房源，掏出手机就有素材。

**捕捉方式**：
- 带看前到 20 分钟，先拍一轮空镜（B-roll）
- 拍 3 张照片：最打动你的空间、最大的问题、窗外景观
- 记 1 句感受："这房子最大的优点是___，最大的坑是___"

**例子**：
| 观察到的东西 | → 视频选题 |
|-------------|-----------|
| 一套房子采光巨好，但户型有硬伤 | 《采光能弥补户型硬伤吗？看完这套你就有答案》 |
| 同一小区两套同户型，差价 200 万 | 《同小区同户型差200万：装修值不值这个溢价》 |
| 房东把房子收拾得特别好 | 《看了一套被房东"养"得很好的房子》 |
| 房子旁边有高压线/垃圾站 | 《这房子什么都好，就是旁边有个…》 |

**关键习惯**：带看 = 拍素材，不是可选项。你已经在花时间去现场了，多拍 5 分钟就多一条视频的素材。

### 3. 手机里的截图 → 数据素材

你每天刷贝壳/链家看到的截图，全是数据素材。

**捕捉方式**：
- 每天刷房源时截 3 张"有意思"的图
- 什么叫有意思？同一小区突然降价的、挂了很久卖不掉的、刚挂就成交的、装修和价格严重不匹配的
- 截图自动存相册，每周整理一次

**例子**：
| 截图内容 | → 视频选题 |
|----------|-----------|
| 同一小区 5 套房源挂牌价对比 | 《同小区挂牌价差200万：房东心态决定报价》 |
| 一套房从1000万降到850万 | 《这套房降了150万还没卖掉，问题出在哪》 |
| 一套房挂3天就成交了 | 《挂3天就成交的房子，做对了什么》 |
| 某板块近3个月成交数据 | 《天山板块Q1真实成交：哪些户型最抗跌》 |

### 4. 同行说了什么 → 对比/反驳素材

同行说的话，不是只能学，还可以**反驳**。

**捕捉方式**：
- 刷到同行的视频/帖子，如果他的观点你觉得不对 → 记下来
- 如果他说的话你觉得"这不就是在忽悠吗" → 这就是一条素材
- 格式：`同行说了什么 + 我为什么不同意 + 真实情况是什么`

**例子**：
| 同行的说法 | → 你的反驳视频 |
|-----------|--------------|
| "这个盘必涨，赶紧入手" | 《有人说这个盘必涨，我给你看三个数据再决定》 |
| "上海房价马上要暴涨了" | 《说暴涨的人，建议你看看这个数据》 |
| "这个小区是长宁最好的" | 《"最好"这个说法我不同意：三个维度重新比》 |

> 注意：反驳不是为了吵架，是为客户提供不同的视角。你的定位是"帮客户看清选项"。

### 5. 评论区和私信提问 → 问答素材

视频号和小红书的评论区、私信里客户问的问题，是最精准的素材。

**捕捉方式**：
- 每条视频发完后，把评论区前 3 个有人问的问题记下来
- 私信里客户问的问题，脱敏后（不提具体客户信息）就是选题
- 一个被问了 3 次以上的问题 → 必须做一条专题视频

**例子**：
| 评论区提问 | → 视频选题 |
|-----------|-----------|
| "改善置换怎么定义？我算改善吗？" | 《你到底算不算改善置换？3个条件自测》 |
| "长宁学区怎么样？" | 《长宁学区真实水平：不吹不黑》 |
| "二手房交易流程是什么？" | 《改善置换全流程：从看房到拿钥匙，每一步要注意什么》 |

### 6. 成交和流失案例 → 故事素材

你做成了一单或丢了一单，都是素材。

**捕捉方式**：
- 每成交或流失一个客户，记一句核心原因
- 成交：客户为什么选这套？决定性因素是什么？
- 流失：客户为什么不买了？去哪了？选了谁？

**例子**：
| 案例 | → 视频选题 |
|------|-----------|
| 客户看了 20 套房最后选了某套 | 《看了20套房最后选了这个：一个改善家庭的决策过程》 |
| 客户嫌贵，去买了更便宜的，后悔了 | 《为了省50万买错房：一个客户的真实教训》 |
| 客户纠结了半年终于下手 | 《为什么他纠结了半年，最后一周就定了》 |

### 7. 日常路过和观察 → 话题素材

你生活在这个城市、这个板块，每天看到的东西就是素材。

**捕捉方式**：
- 路过一个小区，注意它有没有变化（新开什么店、外立面翻新、门口施工）
- 注意到一个现象：比如某个板块突然多了很多中介门店
- 听到一句有意思的话：业主群的讨论、客户闲聊时说的

**例子**：
| 日常观察 | → 视频选题 |
|----------|-----------|
| 某板块新开了一家山姆/盒马 | 《这个板块最近开了家盒马，对房价意味着什么》 |
| 某小区门口突然多了很多房产中介 | 《中介扎堆的板块，是风口还是泡沫》 |
| 下雨天发现某条路积水严重 | 《买这个板块前，下雨天来看一眼》 |

## 素材捕捉系统（每天 5 分钟）

不需要复杂的工具，就用微信文件传输助手 + 手机相册。

### 每日随手记（三种格式任选）

**最简版**：一句话
```
今天带客户看的天山华庭那套，户型有硬伤但是采光巨好 → 可以做一条"采光能弥补户型硬伤吗"
```

**标准版**：一句话 + 一个角度
```
素材：客户问我"为什么天山和古北挨着，单价差3万"
角度：隔一条街差3万，到底差在哪？拆成板块定位/学区/房龄三个维度讲
```

**完整版**：客户原话 + 你的观点 + 画面建议
```
素材：客户看了一套房，进去3分钟就说走
客户原话："说不出来哪里不好，但就是不想待"
我的解读：有些房子"气场"不对——其实是通风、采光、层高三样东西的共同感觉
画面：空房间广角镜头 + 三个数据叠加（通风系数/采光时长/层高）
```

### 每周素材整理（周日花 15 分钟）

打开微信文件传输助手，往上翻这一周随手记的东西，挑出 3-5 条最有感觉的，排下周的拍摄计划。

**判断标准**：
- 这条素材能提炼出一个可争论的观点吗？ → 能 → 用
- 这条素材跟我的定位一致吗？（佳佳 = 改善置换专家）→ 一致 → 优先用
- 这条素材有画面可拍吗？→ 有 → 优先用

## 没素材时的应急方案

如果今天实在不知道拍什么，按顺序试：

1. **翻聊天记录**：打开微信，往上翻最近一周跟客户的聊天，找一句客户问过的问题
2. **刷贝壳 5 分钟**：看最近一周成交数据或挂牌变化，找一个异常的（涨了/跌了/秒卖了/挂很久卖不掉）
3. **翻手机截图相册**：最近一周截过的房源图，找一张有意思的
4. **搜小红书**：搜"上海买房"看最近大家在讨论什么，找一条你不同意或不完全同意的
5. **回顾记忆系统**：读 `memory/layer1-identity/core-thesis.md`，哪个核心观点还没拍过视频 → 今天就拍这个
6. **"我自己"系列**：讲一个你自己买房/看房的真实经历或感悟，第一人称永远不愁没素材

> 这 6 条兜底方案，任何一条在 10 分钟内都能找到一个可拍的选题。

## 常见误区

**误区一："等我想好一个完美的选题再拍"**
→ 真相：没有完美的选题。先拍 10 条一般的，你会从数据反馈里发现什么才是好选题。

**误区二："这个选题太小了，不值得拍一条"**
→ 真相：短视频就是越小越好。"天山vs古北为什么差3万" > "上海买房攻略"。小选题讲得透，大选题讲得泛。

**误区三："这个话题别人拍过了，我再拍没意思"**
→ 真相：同样的选题，你的角度、你的数据、你的表达方式，就是差异化。别人讲"这个小区好不好"，你讲"我带客户看了这个小区，客户说了什么"。

**误区四："我没有专业设备，拍出来不好看"**
→ 真相：房产内容的核心是信息价值，不是画质。一个有用但有噪点的视频 > 一个精美但空洞的视频。
