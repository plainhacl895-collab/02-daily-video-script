# 视频制作运营手册

> 脚本是原材料，制作是烹饪。同样的食材，做法不同，味道天差地别。

---

## 一、封面设计

封面是用户在信息流里看到的**第一帧画面**。划走还是点进来，封面占一半权重。

### 通用原则
- **人脸 > 文字 > 风景**：有人脸的封面点击率高 30%+。如果你出镜，封面必带你的脸
- **一张封面只讲一件事**：不要试图在一张图里塞三个卖点
- **高对比度**：深色背景配白色大字，或浅色背景配深色大字。别让文字和背景融在一起
- **手机端预览测试**：封面在手机上只有拇指大小，缩到 3cm 宽还能看清吗？

### 视频号封面规范
- **尺寸**：竖版 3:4（推荐）或 1:1。视频号信息流里竖版占用面积更大
- **文字限制**：主标题 ≤ 6 个字，副标题 ≤ 10 个字
- **常用布局**：
  - **左字右脸**：左侧 2/3 放文字，右侧 1/3 放你的半身照
  - **大字叠脸**：照片做底，白色大字叠在画面中上位置
  - **纯文字卡片**：深色渐变背景 + 亮色大字（混剪视频常用）

### 小红书封面规范
- **尺寸**：3:4 竖版（小红书全站统一）
- **文字更多**：小红书封面可以承载更多信息，标题可到 15 字
- **风格偏精致**：小红书用户对封面的审美要求更高，避免"中介 style"配色（红底黄字等）
- **常用布局**：
  - 干净的室内照片做底 + 白色/米色文字
  - 数据图表/对比表截图 + 关键数字放大叠层
  - 你本人出镜的半身照 + 一句话核心观点

### 封面文字公式（任选一个）
1. **数字+结果**："1000万，天山vs古北怎么选"
2. **问题+承诺**："看房10分钟就走的房子，长什么样"
3. **反常识+解释**："这户型有硬伤，但我建议买"
4. **标签+筛选**："改善家庭·长宁三选一"

### 封面制作工具
- **剪映**："封面"功能，导出前在剪映里直接做封面，设置封面后导出带封面的视频
- **Canva / 醒图**：需要精细排版时用，做好后导入剪映设为封面
- **快速方法**：视频里选一帧表情自然、光线好的画面截图，叠加 3-6 个字标题

## 二、标题优化

封面决定"看不看"，标题决定"停不停"。

### 通用原则
- **前 15 个字必须出现核心信息**：视频号标题会被折叠，信息流里只显示前 15-20 字
- **数字 > 形容词**："差 200 万" > "差距很大"
- **具体 > 抽象**："天山路 3 个小区对比" > "买房要注意什么"
- **口语化**：标题是你在跟一个人说话，不是写新闻标题

### 标题公式库

| 类型 | 公式 | 示例 |
|------|------|------|
| 数字对比 | X vs Y，差的不只是钱 | "1000万买天山vs古北，差的不是那500万" |
| 反常识 | 都说X，其实Y | "都说这户型完美，我看了3分钟发现一个硬伤" |
| 决策框架 | X个维度，帮你筛出Y | "3个维度，帮改善家庭筛出真·好房子" |
| 客户故事 | 我带客户看了X，结果Y | "我带客户看了一套房，他进去5分钟就说走" |
| 避坑警告 | 买X前，先看Y | "买长宁二手房前，先查这3个数据" |
| 清单式 | X个Y，建议收藏 | "改善置换家庭必查的4个隐性成本" |

### 平台差异化
- **视频号**：标题可以偏"信任型"，强调你的专业判断。"我建议"、"我的看法"可以用
- **小红书**：标题要偏"搜索型"，含关键词。用户在小红书是主动搜的，所以标题里要埋搜索词。比如"上海改善置换"、"长宁看房"、"1000万预算"
- **通用禁忌**：两个平台都不要用"震惊"、"必看"、"手慢无"等标题党词汇——跟人设冲突

### 标题操作细节
- 发之前写 3 个标题备选，选最有"点进去冲动"的那个
- 小红书标题可以带 emoji 作为视觉分隔（📌🏠等），视频号不用

## 三、字幕设计

字幕不是"把说的话打出来就行"——它是画面构成的一部分。

### 字体选择
- **首选**：思源黑体 / 阿里巴巴普惠体（剪映里叫"黑体"即可，清晰、不花哨）
- **禁止**：任何手写体、毛笔体、卡通体（除非极特殊场景）
- **大小**：
  - 常规字幕：字号在剪映里设为 10-12（约占屏幕宽度 1/18-1/15）
  - 强调字幕（关键数字/金句）：字号 18-24，全屏居中 2-3 秒
- **颜色**：白色字 + 黑色描边（可读性最高，不受背景影响）

### 字幕位置
- **常规对话字幕**：屏幕下方 1/5 区域，不遮挡人脸
- **强调关键词**：屏幕居中偏上，停留 2-3 秒后消失
- **数据卡片**：左侧或右侧 1/3 区域，以矩形色块做底

### 强调规则
以下内容必须用**放大字号 + 高亮色**强调：
- 数字（价格、面积、差价、涨幅百分比）
- 金句（全屏居中、2-3 秒停留、语气放慢配合）
- 反转点（"但"、"然而"、"问题是"后面的那句）
- 地名关键词（板块名、小区名、路段名）

### 字幕密度
- 每屏字幕 ≤ 12 个字（多了用户在手机上读不过来）
- 每句字幕停留时间 ≥ 1.5 秒
- 长句断成 2-3 段字幕，按语义断句，不按字数硬断

### 字幕动画
- **入场**：简单淡入（0.1 秒）即可，不用弹跳/旋转/打字机效果
- **强调**：关键词用"放大+缩小"效果（0.3 秒），不要超过 1 次/15 秒
- **金句**：静态居中，不带动画——金句靠内容打人，不靠特效

## 四、剪辑节奏

### 三种形式的剪辑模板

#### 口播剪辑模板

> 以下时间是大致参考，40秒到1分半都行，别太长影响完播率即可。

1. **开头约 3-5 秒**：钩子句 + 你的人脸 + 无BGM（让钩子句干净落地）
2. **展开段**：BGM 淡入，正常节奏，每个句子之间剪掉呼吸停顿（跳剪）
3. **高潮段**：在关键数字/金句处，画面放大到 110%（强调感），BGM 音量降到 20%
4. **收尾段**：恢复，必要时插入一张图片/截图做 B-roll 覆盖（避免全程一张脸）
5. **结尾约 3-5 秒**：BGM 淡出，你的人脸 + 结尾句 + 字幕联系方式叠层

**跳剪规则**：
- 句子之间的"嗯"、"啊"、"就是说"全部剪掉
- 两个句子之间的停顿 ≤ 0.3 秒
- 段落之间停顿 0.5 秒（给观众消化时间）

#### 实拍/探盘剪辑模板

> 总长灵活，40秒到1分半都行。画面本身已传递信息，可以比口播稍长一些。

1. **开头约 3-5 秒**：最有视觉冲击力的空间镜头 + 叠加地点文字（"长宁·天山"）
2. **引入段**：你对镜头开场 + 空间全景镜头交替
3. **主体段**：边走边讲 + 对应空间细节 B-roll，每换一个房间加文字叠层标注
4. **结尾约 5 秒**：最佳空间作为收尾镜头 + 核心金句叠加

**实拍特有规则**：
- 每个空间展示 ≥ 2 个景别（全景 + 细节）
- 走动镜头速度调为 80%（慢一点更稳、更高级）
- 室内转场用"走路遮挡"自然过渡（走到镜头前挡住→切下一个空间），不用花哨转场

#### 混剪剪辑模板

> 总长灵活，卡片承载信息量大，40秒到1分钟通常够用。

1. **开头约 3-5 秒**：标题卡片（大标题 + 副标题）+ BGM 直接进
2. **引入段**：第一张核心数据截图 + VO 开场
3. **主体段**：素材卡片轮播（每张 3-5 秒），关键数字叠层放大
4. **结尾约 5 秒**：总结卡片（3 条要点） + 你的联系方式/引导

**混剪特有规则**：
- 素材之间用"淡入淡出"转场（0.3 秒），不用任何花哨转场
- 每张素材上必须有 1 行文字标注（"2025年Q1成交数据"之类的上下文）
- VO 音量比口播视频大 10%（因为没有画面分散注意力）

### 通用节奏曲线

不管总长多少，节奏按这个感觉走：
```
开头几秒:  钩子 + 无BGM → 冲击力，干净落地
前半段:    BGM淡入 → 建立节奏，信息展开
中段:      核心内容 → 信息密度最高，关键数字/金句处放慢
后半段:    收尾 → 恢复正常节奏，软引导
结尾几秒: BGM淡出 → 干净收尾
```

### BGM 操作细节
- **淡入淡出**：所有BGM必须有 1 秒淡入 + 1 秒淡出，不能硬切
- **人声优先**：任何有口播的段落，BGM 音量 = 人声音量的 25-30%
- **段落间**：两段口播之间的纯画面段落，BGM 音量可以提高到 40%
- **开头钩子**：前 3 秒不用 BGM，让钩子句"裸奔"更有冲击力

## 五、发布时间策略

### 视频号最佳时段（按优先级排列）
1. **周一至周五 12:00-13:00**：午休刷手机高峰，完播率最高
2. **周一至周五 20:00-22:00**：下班后放松时段，流量最大但竞争也大
3. **周六至周日 10:00-12:00**：周末上午悠闲刷手机
4. **避开**：周一早上 8-10 点（通勤时段，注意力分散）、周五晚上 10 点后（出门社交）

### 小红书最佳时段
1. **周一至周五 7:30-8:30**：早起通勤刷小红书
2. **周一至周五 12:00-14:00**：午休时段
3. **周一至周五 18:00-19:00**：下班通勤
4. **周六至周日 9:00-11:00**：周末上午
5. **避开**：工作日下午 2-5 点（都在忙，流量低谷）

### 发布频次策略
- **每周 5-6 条**为最佳频次（验证期可 4-5 条）
- **间隔**：两条之间至少隔 6 小时
- **固定时间**：选定一个时段后尽量固定，算法会给稳定更新的账号加权
- **建议节奏**：周二/四/六/日发，周一/三/五做素材和拍摄

### 发布后操作
- 发布后 **1 小时内**不要关闭 App，保持互动（算法会在发布后 1 小时内测试推荐）
- 自己先评论 1-2 条引导性留言（抛问题、补充观点）
- 发布后 2 小时检查数据：播放量 < 平时 50% → 检查标题/封面是否需要调整；完播率 < 20% → 开头钩子有问题

## 六、标签与话题策略

### 视频号标签
- 每条 3-5 个标签，不要超过 5 个
- **大词 + 小词组合**：`#上海买房`（大词，泛流量）+ `#长宁改善置换`（小词，精准流量）+ `#天山板块`（具体板块）
- 标签放在正文末尾，不用放标题里

### 小红书标签
- 每条 5-8 个标签
- **搜索型标签为主**：小红书的标签本质是搜索关键词，要想"客户会搜什么"
- 标签组合：
  - 2 个泛词：`#上海买房` `#改善置换`
  - 3 个精准词：`#长宁看房` `#1000万预算` `#上海二手房`
  - 1-2 个话题标签：`#我的看房日记` `#买房攻略`
  - 1 个品牌标签（长期）：`#佳佳说房`（建立自己的标签资产）

### 标签操作细节
- 视频号标签用 `#` 号，中间用空格分开
- 小红书标签用 `#` 号，标签之间用空格分开，标签内无空格
- 不用"伪标签"（跟内容无关的热门标签），会被降权

## 七、评论区运营

### 发布后立即操作
1. **自己先发 1-2 条评论**，引导互动方向：
   - 抛问题型："你觉得天山的溢价值不值？评论区聊聊"
   - 补充信息型："这条视频里提到的成交数据，我放在下一期详细拆"
   - 纠正补充型："更正一下，天山华庭不是08年的，是10年的"（如果有错误）
2. **回复前 10 条真实评论**，尽量在 1 小时内回复（算法会记录互动率）
3. **置顶一条引导性评论**（你自己的或优质评论）

### 评论回复原则
- 回复风格跟人设一致：朋友式、不催单、说实话
- 有人问"这个小区怎么样"→ 回答要具体，不敷衍，但私信给细节（"这个说来话长，你可以私信我聊"）
- 遇到杠精不吵架，回复"你说得也对，每个角度看问题不一样"然后停
- 遇到明确买房需求 → 引导私信（"你的预算和需求方便说一下吗？私信我帮你分析"）

### 评论区风控
- 删掉：纯广告评论（"加我微信xxx"）、人身攻击
- 不删：不同意见、合理质疑（放那反而显得真实）
- 标记：频繁出现的关键词/问题 → 记录下来成为下一条视频的选题

## 八、发布前最终检查清单

### 视频文件检查
- [ ] 手机上看一遍完整视频（不止在剪映预览，导出后在相册里看）
- [ ] 字幕检查：有无错别字？关键数字是否加粗/放大？字幕是否遮挡人脸？
- [ ] 音频检查：人声是否清晰？BGM是否盖过人声？开头结尾是否有"咔嗒"杂音？
- [ ] 画质检查：暗光部分有没有噪点？人脸有没有过曝？
- [ ] 结尾检查：联系方式/引导是否包含？（视频号评论引导、小红书评论/私信引导）

### 封面复查
- [ ] 缩到 3cm 宽在手机上还能看清文字吗？
- [ ] 人脸表情自然吗？（不要用说话中截到的奇怪表情）
- [ ] 封面和内容一致吗？（封面别骗人）

### 标题复查
- [ ] 前 15 个字核心信息到位了吗？
- [ ] 有数字/具体信息吗？（不是空泛的形容词）
- [ ] 违禁词扫了一遍吗？（家人们/震惊/重磅/赶紧/手慢无/错过不再/必看/血赚/抢疯了/暴涨/暴跌）
- [ ] 小红书标题里埋了搜索关键词吗？

### 发布操作
- [ ] 标签加好了吗？（视频号 3-5 个，小红书 5-8 个）
- [ ] 发布时间选对了吗？（尽量在推荐时段内发）
- [ ] 发布后自己有 1 小时可以互动吗？

## 九、数据追踪简表

发布后 24 小时、48 小时、7 天各检查一次，记录核心数据。

| 日期 | 标题 | 平台 | 播放量 | 完播率 | 点赞 | 收藏 | 评论 | 分享 | 涨粉 | 有效私信 |
|------|------|------|--------|--------|------|------|------|------|------|----------|
| | | | | | | | | | | |

**核心关注指标排序**（按北极星指标权重）：
1. **有效私信数** ← 北极星，最重要
2. **完播率** ← 内容质量指标
3. **分享数** ← 信任度指标（愿意分享=认可你说的）
4. **收藏数** ← 实用价值指标
5. **播放量** ← 参考即可，不是目标
6. **点赞数** ← 参考即可，点赞最容易但不代表什么

> 播放量高但完播率低 + 0 私信 = 内容没到点上，需要调整角度。
> 播放量一般但完播率高 + 有私信 = 方向对了，继续深挖。
