# 画面布局与标题设计

## 竖屏视频安全区 (1080×1920)

```
┌─────────────────────┐  ← 顶部边缘
│  标题区 (y: 80-280)   │  ← 章节标题、台标
│                     │
│  核心区 (y: 280-1640) │  ← 主要内容，确保不被遮挡
│                     │
│  字幕区 (y: 1640-1820)│  ← 常规字幕，距底 60-100px
└─────────────────────┘  ← 底部边缘
```

## 文字叠加位置速查

| 元素 | y 位置 (px) | 字号 | 对齐 | 持续 |
|------|------------|------|------|------|
| 章节标题 | 80-150 | 60-70 | 左对齐或居中 | 2-3s |
| 强调文字 | 960 (中) | 75-85 | 居中 | 1.5-2.5s |
| 金句 | 960 (中) | 85-100 | 居中 | 2-4s |
| 常规字幕 | 1760-1820 | 55-65 | 居中 | 随语音 |
| 关注引导 | 1760-1820 | 50-55 | 居中 | 2-3s |

## 标题设计原则

1. **短**: 4-8 字最佳，不超过 12 字
2. **对比**: 用颜色/大小与字幕区分
3. **位置**: 固定在顶部或中上部，不跳来跳去
4. **时机**: 段首出现，持续 2-3 秒后消失（不遮挡内容）

## 字幕 vs 标题对比

| 维度 | 字幕 | 标题 |
|------|------|------|
| 字号 | 55-65px | 60-75px |
| 颜色 | 白+黑边 | 金/橙/品牌色 |
| 位置 | 底部 | 顶部/中上 |
| 持续 | 随语音 | 固定 2-3s |
| 字体 | 常规 | 粗体 |

## 色彩使用

- **常规字幕**: 白色 #FFFFFF + 黑色描边（最大对比度）
- **标题**: 金色 #FFD700 或 青色 #00FFFF（醒目但不刺眼）
- **数字强调**: 金色或黄色
- **警告/避坑**: 红色慎用（负面情绪），建议橙色
- **品牌色**: 选一两个，全视频统一

## 背景与文字对比

如果画面背景杂乱（如户外探盘），字幕描边需加粗到 3-4px，或加半透明黑底条。
