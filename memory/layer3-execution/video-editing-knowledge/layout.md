# 画面布局与标题设计

## 知识型内容布局原则

知识型视频的画面承担了比陪伴型更重的信息传递任务——观众在看画面的同时还在消化数据。布局的核心是**分层不打架**。

## 竖屏视频安全区 (1080×1920)

```
┌─────────────────────┐  ← 顶部边缘
│  系列标识区 (y: 40-80) │  ← 系列名/期数（小字，不抢）
│  标题区 (y: 80-280)    │  ← 章节标题、数据卡片
│                     │
│  核心画面区 (y: 280-1560)│  ← 主要内容，数据对比卡片放这里
│                     │
│  字幕区 (y: 1560-1820)  │  ← 常规字幕 + 数据强调字幕
└─────────────────────┘  ← 底部边缘
```

## 知识型内容的四种叠加布局

### 1. 数据卡片（最常用）

关键数字不依赖语音，用卡片形式直接展示：

```
┌─────────────────────┐
│                     │
│   小区实拍/地图       │  ← 背景画面
│                     │
│  ┌───────────────┐  │
│  │ 挂牌均价       │  │  ← 半透明黑底卡片 (70% opacity)
│  │ 108,000 元/㎡  │  │  ← 数字用金色大字
│  │ 天山板块第1    │  │
│  └───────────────┘  │
│                     │
│     [常规字幕]       │
└─────────────────────┘
```

**卡片规则**：
- 每屏最多 1 个数据卡片（不要堆2个以上）
- 卡片停留时间 = 口播说到这个数据的时长 + 0.5秒缓冲
- 卡片背景：黑色 60-70% 不透明度，圆角
- 数字字号 75-90px，单位 40-50px

### 2. 对比表格

同板块小区对比时用：

```
┌─────────────────────┐
│ 天山板块四小区对比    │  ← 表格标题
│ ┌──────┬────┬────┐ │
│ │      │均价 │跌幅│ │
│ ├──────┼────┼────┤ │
│ │新天地│10.8│15% │ │  ← 当前小区高亮行
│ │仁恒  │9.7 │25% │ │
│ │河畔  │9.3 │19% │ │
│ │星城  │9.5 │—  │ │
│ └──────┴────┴────┘ │
└─────────────────────┘
```

### 3. 金句全屏

```
┌─────────────────────┐
│                     │
│                     │
│   房子会旧，         │  ← 宋体 Bold 100px，居中
│   但一个愿意为品质    │
│   投票的社区，       │
│   不容易老。         │
│                     │
│                     │
└─────────────────────┘
```

金句出现时画面做暗角处理（边缘渐暗），让文字成为唯一焦点。

### 4. "适合谁/不适合谁"双栏

```
┌─────────────────────┐
│  适合               │  不适合
│  ✓ 社区纯粹         │  ✗ 要会所
│  ✓ 房龄新           │  ✗ 要学区
│  ✓ 双地铁           │  ✗ 预算<650万
└─────────────────────┘
```

左右分栏，绿色勾/红色叉，一目了然。

## 文字叠加位置速查

| 元素 | y 位置 (px) | 字号 | 字体 | 对齐 | 持续 |
|------|------------|------|------|------|------|
| 系列标识 | 40-80 | 35-40 | 黑体 Regular | 左/右上角 | 全程或段首3s |
| 章节标题 | 100-180 | 60-70 | **宋体 Bold** | 居中 | 2-3s |
| 数据卡片 | 400-900 | 75-90(数字) | 黑体 Medium | 居中 | 随数据口播 |
| 全屏金句 | 960 (中) | 90-110 | **宋体 Bold** | 居中 | 口播+停留1.5s |
| 常规字幕 | 1680-1800 | 58-65 | 黑体 Regular | 居中 | 随语音 |
| 数据强调 | 随字幕位 | 70-80 | 黑体 Medium | 居中 | 随数据口播 |
| 关注引导 | 1680-1800 | 45-50 | 黑体 Regular | 居中 | 2-3s |

## 系列视频的统一元素

知识型内容是系列化的。每个系列建立统一的视觉识别：

- **系列标识**：固定位置（建议左上角），小字显示"天山板块小区系列"
- **期数标记**：右上角或系列标识旁，"第1期"
- **统一字体方案**：整个系列用同一套字体搭配
- **统一BGM**：整个系列用同一首或同风格的BGM

## 色彩使用

| 用途 | 颜色 | 色值 |
|------|------|------|
| 常规字幕 | 白色 | #FFFFFF |
| 数据数字 | 金色 | #FFD700 |
| 正面/优点 | 暖绿 | #4CAF50 |
| 负面/缺点 | 暖橙（不用红色） | #FF8C00 |
| 卡片背景 | 黑 70% | rgba(0,0,0,0.7) |
| 金句背景 | 画面暗角 | 径向渐变 |

## 背景与文字对比

如果画面背景杂乱（如户外探盘），字幕描边需加粗到 3-4px，或加半透明黑底条。知识型内容的户外场景多（小区外立面、周边配套街拍），要注意浅色天空背景下的白色字幕可读性。
