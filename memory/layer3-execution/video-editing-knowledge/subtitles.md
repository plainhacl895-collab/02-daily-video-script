# 字幕设计原则

## 时间同步铁律

**字幕出现**: 说话人第一个字发声的同时 → 字幕出现
**字幕消失**: 说话人最后一个字结束的同时 → 字幕消失（可延长 0.1-0.2s 缓冲）

技术实现: 使用 Whisper 词级时间戳而非估算。每个字幕行的 start = 该行第一个词的时间戳，end = 该行最后一个词的时间戳。

## 断行规则

| 条件 | 处理 |
|------|------|
| 一行 ≤ 18 个中文字 | 单行显示 |
| 一行 > 18 个字 | 在标点处断为两行 |
| 英文/数字混排 | 按字符宽度等效计算（1 中文 ≈ 2 英文） |
| 专有名词 | 不断开（如"新天地河滨花园"不跨行） |

## 安全区

```
┌─────────────────────┐
│   标题安全区 (20%)    │  ← 章节标题、台标
│                     │
│   内容安全区 (60%)    │  ← 强调文字、金句
│                     │
│   字幕安全区 (20%)    │  ← 常规字幕
└─────────────────────┘
```

字幕距底部至少 60px（约占屏高 3%），确保在小屏手机上不被遮挡。

## 样式默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 字体 | 思源黑体 Regular | 现代、清晰 |
| 字号 | 58px | 竖屏 1080p 推荐 |
| 颜色 | 白色 #FFFFFF | 最高对比度 |
| 描边 | 黑色 2px | 浅色背景可读 |
| 阴影 | 1px | 增强立体感 |
| 位置 | 底部居中 (alignment=2) | 行业标准 |

## 特殊场景

- **数字强调**: 数字单独一行、加大字号、改颜色（如金色）
- **金句**: 全屏居中、最大字号、可加淡入动画
- **问答**: 问题用不同颜色或前缀标记（如 Q: / A:）
- **中英混杂**: 英文用半角空格隔开，保持可读性
