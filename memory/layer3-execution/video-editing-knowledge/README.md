## v3.0 核心变化（2026-05-23）

从陪伴型→知识型转型，全局原则调整：

1. **节奏更慢**：BGM更轻、淡入淡出更长、金句停留更久。给观众思考的时间
2. **信息更重**：数据不会只靠嘴说，必须在画面上同步出现（数据卡片、对比表格）
3. **气质更沉**：BGM从"轻快现代"→"舒缓钢琴"，字体从纯黑体→黑体+宋体搭配，音效从"点缀"→"几乎不用"
4. **系列感**：统一字体/BGM/标识位置，一期一期建立视觉识别
5. **金句升维**：不只是功能总结，要有文化气息。全屏宋体+画面暗角+说完停留

## 适用内容类型

本知识库为以下内容类型服务：
- **小区深度分析**（主打）：天山板块小区系列、古北系列、中山公园系列
- **板块对比**：天山 vs 古北 vs 中山公园
- **数据解读**：成交数据、价格走势、市场分析
