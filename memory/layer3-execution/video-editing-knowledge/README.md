按需查阅，不预加载。每个文件聚焦一个主题。

## 核心原则

1. **字幕同步 > 字幕美观**：时间不准，再好看也没用
2. **字体是衣服**：不同内容类型配不同字体气质
3. **音效是标点**：强调关键信息，不是每句都加
4. **BGM是底色**：不抢人声，提供情绪框架
