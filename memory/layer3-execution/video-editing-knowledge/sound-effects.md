# 音效使用指南

## 知识型内容的音效原则

**宁缺毋滥。** 知识型内容和泛娱乐内容的音效使用有本质区别：

| | 泛娱乐/陪伴型 | 知识型 |
|------|------|------|
| 音效频率 | 3-6个/分钟 | **1-2个/分钟** |
| 音效类型 | pop、ding、whoosh | 极简的强调音、氛围音 |
| 情绪目标 | 轻松、有趣 | 沉静、可信 |
| 风险 | 太干 | **太吵 → 廉价感** |

**铁律**：每多一个音效，知识型内容的"文化感"就减一分。如果你不确定要不要加音效，答案是"不加"。

## 知识型内容适用的音效

| 场景 | 音效类型 | 使用时机 | 说明 |
|------|----------|----------|------|
| 数据出现 | 极简的钢琴单音或轻微的"叮" | 关键数字在画面上弹出的同时 | 只用于最重要的1-2个数字，不要每个数字都加 |
| 段落转换 | 极轻的氛围音（pad/atmosphere） | 从"优点"切到"缺点"这类大段落转换时 | 不是 whoosh，是 0.5-0.8 秒的氛围过渡 |
| 金句出现前 | 留白（无声） | 金句出现前 0.3-0.5 秒 | **无声比有声音效更有力**——突然的安静制造注意力集中 |
| 章节标记 | 极其克制的提示音 | 章节标记出现的瞬间 | 可选，不强求 |

## 知识型内容不用的音效

- **comedy/cartoon 类**：太轻浮，和文化感冲突
- **whoosh/swoosh 转场**：太"自媒体模板感"
- **pop/ding 高频**：太闹，打断思考
- **任何连续出现超过2次的音效**：重复 = 廉价

## 使用频率

| 视频时长 | 知识型推荐音效数 | 最多不超过 |
|----------|-----------------|-----------|
| 30-40 秒 | 0-1 个 | 2 个 |
| 40-60 秒 | 1-2 个 | 3 个 |
| 60-90 秒 | 2-3 个 | 4 个 |

**如果整条视频一个音效都不用，也是完全OK的。** 好的BGM + 好的字幕设计已经足够撑起知识型内容的节奏感。小光头的视频就是一个音效都不用但节奏依然成立的典范——他的节奏来自文案本身的张弛，不是靠音效砸出来的。

## FFmpeg 音效混入

```bash
# 在 12.5 秒处混入一个 0.3 秒的音效
ffmpeg -i video.mp4 -i effect.wav \
  -filter_complex "[1:a]adelay=12500|12500,volume=0.4[fx];[0:a][fx]amix=inputs=2:duration=first" \
  -c:v copy output.mp4
```

注意知识型内容的音效音量比常规低：volume=0.4 而非 0.6。音效应该"感觉到"而不是"听到"。
